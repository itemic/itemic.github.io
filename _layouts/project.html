VOXSpell

VOXSpell is a spelling practice application made using Java and JavaFX. It runs on macOS and makes use of the built-in speech library. It also runs on Linux with the Festival speech library installed. VOXSpell is fully customizable and supports custom word lists.

VOXSpell was developed as the final project for SOFTENG 206.

Task Scheduler (Revilo)

Revilo is a task scheduler, which takes tasks given in the form of a DAG and outputs the optimal schedule on a specified number of processors, using a branch-and-bound algorithm. This project was done in Java, and the GUI was designed using the JavaFX GUI framework.

Revilo was developed as the final project for SOFTENG 306.
